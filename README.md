Leon is a self-hosted AI orchestrator that provides voice-controlled management of multiple Claude Code agent instances. Features a cyberpunk-themed dashboard for monitoring and controlling agents, voice command input, and coordinated multi-agent task execution.

## Features

### Voice Control
- Voice command input for hands-free agent management
- Natural language task dispatch

### Multi-Agent Management
- Spawn and manage multiple Claude Code instances
- Coordinated task execution across agents
- Agent status monitoring and output streaming

### Dashboard
- Cyberpunk-themed web dashboard
- Real-time agent status and output
- Task queue visualization
